# BLDC-Simulation-
Sensored BLDC motor control using 16F877 Proteus simulation 
Proteus Design Suite : 8.9 SP2
Microprocessor       : [PIC16F877] 
Drive IC             : [TC4469]
MOSFET               : P channel - IRF5305
MOSFET               : N channel - IRLI3705
Speed reg            : resistive
